此腳本靈感取自 [9nice](https://greasyfork.org/users/970328)的[禁漫天堂-方向鍵快速切換上下一話](https://greasyfork.org/scripts/453029)  <br/>
並新增翻頁功能 <br/>

功能介紹： <br/>
Esc 返回目錄 <br/>
Shift + ([←] 或 [A]) 切換上一話 <br/>
Shift + ([→] 或 [D]) 切換下一話 <br/>
[←] 或 [A] 切換上一頁 <br/>
[→] 或 [D] 切換下一頁 <br/>
